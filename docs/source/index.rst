ByteFight Developer Documentation
===================================

**ByteFight** is a game AI competition hosted by Georgia Tech. This documentation is meant to serve
as a living reference for new and old developers alike to understand the architecture and
design principles of the different parts of the competition.

Check out the :ref:`documentation-page` section to find out how to contribute to documentation.

.. note::

   This project is under active development.
